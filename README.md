ExpenseTracker

A personal finance application built with React that allows users to track income and expenses, view a summary of their balance, and visualize spending with a bar chart.

Live Demo
https://expense-tracker-pi-liart.vercel.app/

Features
- Add and delete income and expense transactions
- Real-time balance, income and expense summary
- Data visualization with an interactive bar chart
- Transactions persist across sessions using localStorage
- Dark glassmorphism UI
- useReducer and Context API for global state management

Tech Stack
- React 18
- Custom Hooks (useLocalStorage)
- useReducer and Context API
- Recharts
- CSS Modules
- Deployed on Vercel

Run Locally
git clone https://github.com/m-ahmad-12/expense-tracker.git
cd expense-tracker
npm install
npm run dev
